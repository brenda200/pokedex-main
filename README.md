# 🐾 Pokedex Digital

Uma **Pokedex digital** que exibe informações sobre diferentes Pokémon, incluindo seus tipos, habilidades e outros detalhes. Este projeto foi desenvolvido para aprender mais sobre integração de dados e exibição interativa.

🔗 [Veja a Pokedex online](https://brenda200.github.io/pokedex-main/)

## ✨ Funcionalidades

- 🔍 Pesquisa por nome ou ID do Pokémon
- 🐉 Exibição de dados como tipo, habilidades, estatísticas e mais
- 📱 Layout responsivo para dispositivos móveis
- 🎨 Design inspirado no estilo clássico da Pokedex

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (para manipulação dos dados e interação)
- API (possivelmente de dados dos Pokémon, como PokéAPI)
